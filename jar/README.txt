Os arquivos configPotigol.zip e JCEditor.jar devem estar no mesmo local durante a primeira
execução do programa. Após isso, o arquivo configPotigol.zip pode ser deletado.

Se você estiver usando o Linux, execute o editor pelo Terminal com o comando java -jar JCEditor.jar.
Se você estiver utilizando a versão 1.9 ou anteriores, entre no Terminal e permissões ao arquivo
.sh (responsável pela execução do Potigol), para isso, entre na pasta ConfigJCE/.potigol
(localizada em sua pasta de usuário) e depois digite chmod 777 ExecPotigol.sh
